# [1] overview

## 개발 목표 및 배경

- 비동기성 경쟁방식 개발
- 기본적인 언어구사를 위한 단어 암기의 중요성
- 단어 암기에 있어서 지루함이라는 장벽 존재
- 스마트폰의 대중화

## 기대 효과

- 경쟁을 통한 영어에 대한 흥미 유발
- 단어 암기에서 비롯되는 지루함 감소
- 단순한 단순 학생들 뿐 아니라 폭 넓은 연령대로 고객 층 확대
